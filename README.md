# 🐦 Flappy Bird – Android (Kotlin)

This project is a simple Flappy Bird-style game developed using Kotlin in Android Studio.

## 🎯 Purpose

The main goal of this project was to understand:

* Kotlin basics and syntax
* Android project structure
* XML layouts and UI components
* Activity lifecycle
* Debugging and debug mode
* Using Android Emulator
* Generating APK files
* App configuration (app name, icons, manifest)
* Gradle build system basics

## 🛠 Technologies Used

* Kotlin
* Android Studio
* XML for UI Design
* Gradle Build System

## 🚀 Features

* Basic Flappy Bird gameplay
* Obstacle movement
* Score tracking
* Game restart functionality

## 🕹 How to Use the App

### ▶️ Running the App (Development Mode)

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/flappy-bird-android.git
   ```

2. Open the project in **Android Studio**.

3. Let Gradle sync completely.

4. Connect a physical Android device
   OR
   Start an Android Emulator.

5. Click **Run ▶️** in Android Studio.

---

### 📦 Installing APK

1. Locate the generated APK file:

   ```
   app/build/outputs/apk/debug/app-debug.apk
   ```

2. Transfer it to an Android device.

3. Enable:

   ```
   Install from Unknown Sources
   ```

4. Tap the APK file and install.

---

### 🎮 Gameplay Instructions

* Tap the screen to make the bird jump.
* Avoid hitting the pipes.
* Each pipe passed increases your score.
* If you collide, the game ends.
* Restart to try again and beat your high score.
